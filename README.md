# Stellaris · 游戏级群星运行模拟器

以视觉沉浸与沙盒创造为核心的游戏化太空模拟器。在物理自洽的引力世界里创造恒星、行星与黑洞，观察它们在感知时间下共舞——一个看得爽、玩得转的「宇宙游乐场」。

纯静态前端，零后端，可离线运行。

**🌌 在线预览： https://moss-550-w.github.io/Stellaris/**

---

## 核心特性

- **物理自洽**：原生 TS 在 Web Worker 中运行 N 体引力，三档精度——
  - 流畅模式（半隐式辛欧拉）支撑更多天体；
  - 标准模式（velocity Verlet）高精度，能量漂移 < 0.01%，为成就判定基准；
  - 实验模式（RK4 四阶）守恒量诊断 HUD（能量/角动量实时漂移），经典周期解（8 字三体、拉格朗日 L4·L5）+ CSV 报告导出。
- **天体演化**：恒星按质量与年龄走完生命周期——主序 → 红巨星 → 白矮星 / 中子星 / 黑洞，大质量恒星触发超新星，演化倍率独立可调。
- **航天器与探测器**：一键发射受控载具（测试粒子），顺/逆行推进做轨道机动与引力弹弓，燃料即 Δv 预算，飞掠天体解锁科普。
- **感知时间**：1× ≈ 30 秒/年，档位 暂停 / 0.1 / 1 / 10 / 100 / 1000 / 10000×，高速自动降精度并提示「细节省略」。
- **可玩沙盒**：点击拾取天体，实时编辑质量/速度/半径/材质/颜色；增删天体、轨道预测、撤销/重做（完整快照，上限 20 步）。
- **三级碰撞**：低速合并 / 中速撕裂 / 高速毁灭，材质影响结果，黑洞接触即吞噬。
- **影视级视觉**：程序化恒星/气态/岩质纹理、大气光晕、黑洞吸积盘与**引力透镜**；合并后处理单通道 Shader（bloom + ACES 色调映射 + 暗角）。
- **在线分享（纯前端）**：一键生成自包含分享链接（deflate 压缩 + base64url），打开即加载他人宇宙；内置官方场景库，零后端、离线可用。
- **成就/挑战**：精确目标 + 实时透明进度（如「宜居双星」），宜居带可视、引力弹弓能量计。
- **沙盒工具**：多预设、存档/读档、JSON 导出导入、截图、新手引导、科普卡。
- **性能可控**：三档画质，移动端自动降档触控适配。

## 技术栈

| 领域 | 选型 |
|------|------|
| 构建 | TypeScript + Vite 8 |
| 渲染 | Three.js（WebGL） |
| 后处理 | 自研合并单通道 Shader（GLSL） |
| 物理 | 原生 TS（Web Worker），天文单位制 G = 4π²，三档积分（欧拉/Verlet/RK4） |
| UI | Vue 3 |
| 存储 | LocalStorage + File API |
| 分享 | 原生 CompressionStream（deflate）+ base64url，纯前端自包含 |
| 部署 | 纯静态资源，零后端，GitHub Actions 自动发布 Pages |

## 架构

五层松耦合，物理与渲染完全解耦（双缓冲快照 + Transferable 零拷贝，一阶外推插值消除通信延迟）。详见 [`CLAUDE.md`](./CLAUDE.md) 与 [`plan.md`](./plan.md)。

```
UI 数据层 (Vue) → 玩法交互层 → 渲染表现层 (主线程 Three.js)
                                    ↑ 双缓冲 + 外推
                              物理仿真层 (Web Worker)
                                    ↑
                              基础内核层 (数学/向量/随机)
```

## 开发

```bash
npm install      # 安装依赖
npm run dev      # 本地开发（http://localhost:5173）
npm run build    # 类型检查 + 生产构建（产物在 dist/）
npm run preview  # 预览生产构建
npm run typecheck
```

环境要求：Node 18+。

## 部署

构建产物为纯静态资源（`vite.config.ts` 中 `base: './'` 使用相对路径，兼容子路径与离线）：

- **GitHub Pages（已配置自动部署）**：`.github/workflows/deploy.yml` 在推送 `main` 时自动 `npm ci && npm run build` 并发布 `dist/` 至 Pages，线上地址 https://moss-550-w.github.io/Stellaris/ 。
  - 首次启用：仓库 **Settings → Pages → Build and deployment → Source** 选择 **GitHub Actions** 即可。
- **Vercel / Netlify**：构建命令 `npm run build`，输出目录 `dist`。
- **离线**：任意静态服务器托管 `dist/` 即可运行。

## 操作指引

- **视角**：拖拽旋转 · 滚轮缩放 · 右键平移（移动端单指旋转、双指缩放）。
- **选中**：点击天体打开编辑器与科普卡；编辑即时生效。
- **调速**：底部时间条；高速观察长期演化。
- **挑战**：左侧面板选择目标，切换「标准」精度开始判定。
- **实验**：右上精度切「实验」启用 RK4 与守恒量诊断 HUD，可导出 CSV 报告；配合场景库的「8 字三体 / 拉格朗日」观测经典周期解。
- **分享 / 场景库**：左上 ☰ 菜单「🔗 分享链接」生成自包含链接；「🌌 场景库」一键加载内置精选宇宙。
- **菜单**：左上 ☰ 存档/读档、导出导入、截图、重看引导。
