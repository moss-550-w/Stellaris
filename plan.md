## 1. 项目总览

**定位：** 以视觉沉浸与沙盒创造为核心的游戏化太空模拟器，给玩家一个物理自洽、视觉震撼、操作自由的「宇宙游乐场」。

**核心体验（五大原则）：** 体验优先 · 物理自洽 · 视觉震撼 · 交互自由 · 性能可控。

**技术栈速查：**

| 领域 | 选型 |
|------|------|
| 构建 | TypeScript + Vite |
| 3D 渲染 | Three.js（WebGL） |
| 后处理 | 自研合并后处理 Shader（单通道） |
| 物理引擎 | 原生 TS（Web Worker），标准模式 Verlet；**不引入 Pyodide** |
| UI | Vue 3 |
| 存储 | LocalStorage + File API |
| 部署 | 纯静态资源，零后端（GitHub Pages / Vercel，可离线） |

**五层松耦合架构（物理与渲染完全解耦）：**

```
┌─────────────────────────────────────┐
│           UI 数据层                  │  Vue 3 面板、存档、设置
├─────────────────────────────────────┤
│          玩法交互层                  │  视角、编辑、成就、沙盒工具
├─────────────────────────────────────┤
│          渲染表现层 (主线程)          │  Three.js 3D场景、粒子、特效
│  ← 双缓冲+预测插值获取位置 →         │
├─────────────────────────────────────┤
│          物理仿真层 (Web Worker)     │  动力学引擎、碰撞、演化、快照
├─────────────────────────────────────┤
│          基础内核层                  │  数学库、矢量运算、随机生成器
└─────────────────────────────────────┘
```

**关键架构规则速记：**
- 物理计算与渲染完全解耦，通过结构化消息通信。
- 物理 Worker 维护**双缓冲**（一计算、一传输），渲染线程取最新缓冲做**一阶外推插值**，消除通信延迟。
- 撤销/重做基于**完整系统快照**（最多 20 步），含所有天体状态与随机种子。
- 所有半透明对象按**显式排序管线**渲染。

**总周期：** 20 周 · **团队假设：** 1~2 名全栈 + 1 名技术美术。

## 3. 阶段一：核心原型验证 ✅ 已完成（2026-06-20）

**目标：** 单恒星系统 Demo，跑通「物理 → 渲染」主链路与感知时间系统。

**单位制：** 天文单位制（AU · 年 · 太阳质量），G = 4π²，地球轨道周期恰为 1 年。

**任务清单：**

**物理内核**
- [x] velocity Verlet 全 N 体积分器（`physics/integrator.ts` ＋ `constants.ts`），恒星 + 4 行星圆轨道。
- [x] 感知时间系统（`core/time.ts`）：1× = 30 秒/年；档位 暂停/0.1/1/10/100/1000/10000，映射真实步长，隐藏真实倍率。
- [x] 物理 Worker accumulator 主循环（`physics/worker.ts`）：固定步长积分、高速降精度(细节省略)、快照 Transferable 零拷贝回传。

**渲染表现**
- [x] 主线程渲染循环：基于「最新快照 + 速度」做**一阶外推插值**（`SimulationController.onFrame`），消除通信延迟与帧率差。
- [x] 自由相机 OrbitControls（轨道/平移/缩放 + 阻尼）。
- [x] 岩质行星材质：程序化 fBm 噪声基底 map + bumpMap，受恒星点光源照亮（`proceduralTexture.ts` / `CelestialFactory.ts`）。〔预制特征遮罩留待阶段三〕
- [x] 恒星自发光球 + 加性光晕 sprite + 点光源；程序化星空点云背景。
- [x] 行星轨迹线（环形缓冲，增强「运行」观感）。

**玩法 / UI**
- [x] 调速面板（`ui/ControlPanel.vue`）：档位/暂停切换、模拟年数、感知速率描述、细节省略徽标、FPS、Worker 连通指示。

**交付物：** 单恒星系统在感知时间下稳定公转，自由视角观测，岩质行星受光可见，调速实时生效。

**验收结果（全部达成）：**
- `vue-tsc --noEmit` 零错误；`vite build` 成功（30 模块，worker 独立打包）。
- 物理正确性（独立脚本核验）：10 年模拟**能量漂移 0.000000%**（远优于标准模式 <0.01% 目标），轨道半径波动 0.0142%，轨道近乎完美闭合。
- `npm run dev` 启动无报错；首页与全部关键模块（含 `.vue`、worker、控制器、渲染工厂）transform 均返回 HTTP 200。
- 物理与渲染已彻底分离（Worker ↔ 主线程，结构化消息 + Transferable）。

## 4. 阶段二：可玩沙盒核心（5 周 · W5–W9）

**目标：** N 体、碰撞、天体编辑器、撤销/重做、轨道预测，形成可玩沙盒。

**任务清单：**

**物理**
- [ ] 全 N 体引力计算，两档精度：
  - 娱乐流畅模式（默认）：半隐式欧拉 ＋ 远距合并，支持 **200+ 天体**
  - 平衡标准模式（进阶）：Verlet 全 N 体精确，支持 **80+ 天体**，千年能量漂移 **<0.01%**（成就/挑战判定基准）
- [ ] 高速（>1000 倍）自动降低物理帧率 ＋「细节省略」提示。
- [ ] 碰撞三级判定：
  - 低速合并（<0.5 倍逃逸速度）
  - 中速撕裂（0.5~1.5 倍逃逸速度）
  - 高速毁灭（>1.5 倍逃逸速度）
  - 材质影响结果（气态易合并 / 岩石易碎裂 / 恒星碰撞触发超新星）；含洛希极限计算。
- [ ] 轨道预测（前瞻积分输出轨迹线）。

**玩法 / 状态**
- [ ] 天体编辑器（质量 / 速度 / 位置 / 材质实时编辑，即时生效可见）。
- [ ] 预设场景加载。
- [ ] 撤销/重做：基于**完整系统快照**（含全部天体状态 ＋ 随机种子，最多 20 步），回滚整盘状态。

**架构**
- [ ] Web Worker 物理/渲染彻底分离落地（结构化消息通信定型）。

**交付物：** 可自由增删/编辑天体、观察碰撞与轨道预测、撤销/重做整盘状态的沙盒。

**验收标准：** 流畅模式 200+ 天体、标准模式 80+ 天体达标；快照回滚正确；碰撞三级判定符合速度阈值。

**性能**
- [ ] 桌面 60fps / 移动 30fps 调优；多档画质与物理精度切档。

**验收标准：** 桌面稳定 60fps；透明排序无深度错误；引力透镜按角度策略正确；成就/挑战判定透明可见。

## 7. 横切关注点（贯穿全程的硬约束）

每阶段自查，任一违背需回退讨论：
- [ ] 主线程不承载重计算（重计算一律入 Worker）。
- [ ] 时间相关逻辑一律映射感知时间系统。
- [ ] 半透明对象一律走显式排序管线，不可省略。
- [ ] 视觉优先混合纹理策略，不承诺纯程序化（气态 / 恒星除外）。
- [ ] 物理精度提升属 V2.0 范围，V1.0 不扩张。
- [ ] 所有修改保证向下兼容，不随意改动原有正常逻辑。
